# Random Color Generator

This is a small web application that generates random colors and displays them on the page. Built using HTML, CSS, and JavaScript, it’s a fun and interactive way to explore colors dynamically.

- Features: 
1. Generates a new random color on button click.
2. Displays the color name in RGB format.
3. Changes the background color of the display box to the generated color.
4. Clean and responsive design.

- Demo

Open the html file in your browser to try it out locally.
You can click the “Generate color” button multiple times to see new colors.

- Technologies Used

1. HTML5 – Page structure and elements.
2. CSS3 – Styling the layout, button, and color display box.
3. JavaScript (ES6) – Generating random colors and updating the UI dynamically.

- How It Works

The button triggers a click event using JavaScript.
When clicked, the getRandomColor() function generates random RGB values.
The generated color is displayed as text in the h2 element. 
The div background color changes to the generated color, giving a live preview.
