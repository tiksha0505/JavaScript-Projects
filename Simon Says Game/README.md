# Simon Says Game

A browser-based Simon Says game built with HTML, CSS, and JavaScript. Test your memory and try to repeat the color sequences!

- Features:
1. Interactive Simon Says memory game.
2. Four colored buttons: Pink, Blue, Yellow, Orange.
3. Tracks current level and highest score.
4. Visual feedback for correct and incorrect moves.
5. Responsive design and smooth button animations.

- Demo:

Open html file in a browser and press any key to start the game. Click the colored buttons in the sequence shown to continue.

- Technologies Used:

1. HTML5 – Page structure and game layout.
2. CSS3 – Styling buttons, colors, animations, and game interface.
3. JavaScript (ES6) – Game logic, sequence generation, event handling, and scoring.

- How It Works:
1. Press any key to start the game.
2. The game randomly highlights a color button.
3. Repeat the sequence by clicking the buttons in the correct order.
4. Each correct sequence increases the level by 1 and adds a new random color to the sequence.
5. A wrong input shows Game Over and displays your score.
6. Highest score is tracked until the page is refreshed.
